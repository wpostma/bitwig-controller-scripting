LINE6 POD XT LIVE

Important note: For the script to work, you need to set up the first 8 banks of the podgo to have amp, stomp, mod and delay mode OFF for the first 8 banks, of four presets. So thats 32 presets to go in and set up manually, or you can try to get this set up via the Line6 app. It's probably easier to just do this by hand. Basically you just go to bank 1, press A, then if AMP or STOMP or MOD or DELAY is lit, press it again so it's not lit, then press SAVE. Advance from 1A, 1B,1C,1D up to 8D.
